Description for Project 1 madafaka 温度変換のウェブアプリを作成しました。
カ氏温度からセ氏温度、または その逆に温度を変換することができます。
温度を変換した後はテキストボックスに数字入力できないようになっていて、Clearボタンを押すとボックス内の文字を消して、また新しい数字を入力することができます。
また、もし数字以外の文字を入力して変換ボタンを押すとエラメッセージが表示されるようになっています。

Description for Project 2 madafaka Chomp Gameという簡単なゲームを作成しました。
Chomp Gameは縦6列、横7列に並べられたボタンを消していって最後の一つを消した人が勝ちというゲームです。
一番左下の赤いボタンは選択できないようになっており、自分が選択したボタンの上側から右側にかけて90度のエリアが消えるようになっているので、一つずつ消していくことも一度に何個ものボタンを消すこともできます。
最初はPlayer 1から始まります。
